# 생성
* vulcanjs-cli(https://github.com/VulcanJS/vulcanjs-cli) 사용
```
vulcan create sheepchat

cd sheepchat
npm install
```
* settings.json 만들기
```
mv sample_settings.json settings.json
```

* 초기화
```
git init
```

* package 삭제
.meteor/packages 에서 example-simple 을 주석처리

* directory 생성
  * components
	* models
	* routes
	* lib
# 실행
npm start
